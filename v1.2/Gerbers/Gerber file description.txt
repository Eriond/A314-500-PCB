The following files are supplied, and should be stacked as follow (top to bottom):

GKO : Board outline information

GTO : Top silkscreen
GTS : Top soldermask
GTL : Gerber top layer
G2L : Gerber first inner layer (Ground plane)
G3L : Gerber second inner layer (Power plane and some signals)
GBL : Gerber bottom layer
GBS : Bottom soldermask
GBO : Bottom silkscreen

TXT : Excellon Drill data
